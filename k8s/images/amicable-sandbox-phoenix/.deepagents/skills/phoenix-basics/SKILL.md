# Phoenix Basics

## When To Use
- You are editing the Phoenix template.

## Conventions
- Keep controllers thin and move shared logic to contexts.
- Prefer explicit route names and small, testable modules.

## Preview
- The sandbox preview runs `mix phx.server` on port 3000.
