# Amicable Documentation

Amicable is an AI-powered web app builder. Users chat with an agent in a React editor, and the agent edits code inside isolated Kubernetes sandboxes with live preview updates.

This documentation is organized for day-to-day development, deployment, and operations.

## Documentation Map

- [Usage](usage.md): Product behavior, user flow, and core concepts.
- [Development](development.md): Local setup, workflows, and codebase layout.
- [Architecture](architecture.md): Components, runtime flow, and message protocol.
- [Configuration](configuration.md): Environment variables and runtime toggles.
- [Deployment](deployment.md): Kubernetes deployment paths and release flow.
- [Testing](testing.md): Test, lint, typecheck, and validation workflows.
- [Operations](operations.md): Runbook and incident response basics.
